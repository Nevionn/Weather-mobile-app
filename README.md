# Установка из исходников

```bash
git clone https://gitlab.com/web4450122/weather-mobile-app.git

cd weather-mobile-app

npm install
```

## Запуск на андроид эмуляторе 

```bash
npx react-native run-android
```

## Запуск на реальном андроид устройстве

```bash
npm run android
```
